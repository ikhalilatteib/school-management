
## Simple school management API


### Installation

```bash
git clone https://github.com/ikhalilatteib/school-management
```

```bash
cd school-management
```
```bash
composer install
```
```bash
cp .env.example .env
```
```bash
php artisan key:generate
```
```bash
php artisan migrate --seed
```

### Default user

```bash
email: admin@app.com
password: password
```
